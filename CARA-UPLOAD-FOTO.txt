=====================================
PANDUAN UPLOAD FOTO PROFIL - STEP BY STEP
=====================================

LANGKAH 1: DOWNLOAD FOTO DARI WHATSAPP
---------------------------------------
1. Buka WhatsApp Web di browser
2. Cari gambar/foto yang ingin Anda gunakan
3. Klik pada gambar tersebut (akan membuka gambar besar)
4. Klik kanan pada gambar
5. Pilih "Save image as..." atau "Simpan gambar sebagai..."
6. Pilih lokasi: Desktop (agar mudah ditemukan)
7. Ubah nama file menjadi: profile.jpg
8. Klik "Save" atau "Simpan"

LANGKAH 2: BUKA FOLDER PROJECT
-------------------------------
Folder project Anda ada di:
d:\jule\portopolio\images\profile\

Cara membuka:
1. Tekan tombol Windows + E (untuk membuka File Explorer)
2. Di address bar, ketik atau copy-paste: d:\jule\portopolio\images\profile\
3. Tekan Enter
4. Folder akan terbuka (folder ini masih kosong)

LANGKAH 3: COPY FOTO KE FOLDER
-------------------------------
1. Buka folder Desktop (tempat Anda menyimpan profile.jpg tadi)
2. Cari file profile.jpg
3. Klik kanan pada file profile.jpg
4. Pilih "Copy" atau "Salin"
5. Kembali ke folder d:\jule\portopolio\images\profile\
6. Klik kanan di area kosong folder
7. Pilih "Paste" atau "Tempel"
8. File profile.jpg sekarang ada di folder images/profile/

LANGKAH 4: BERITAHU SAYA
-------------------------
Setelah file berhasil dicopy, beritahu saya dengan mengetik:
"Sudah selesai, nama filenya: profile.jpg"

Saya akan langsung update kode HTML untuk menampilkan foto Anda!

=====================================
ALTERNATIF CARA YANG LEBIH MUDAH:
=====================================

CARA DRAG & DROP:
1. Download foto dari WhatsApp ke Desktop (seperti Langkah 1 di atas)
2. Buka folder d:\jule\portopolio\images\profile\
3. Buka folder Desktop di window lain
4. DRAG (tarik) file profile.jpg dari Desktop
5. DROP (lepas) ke folder images/profile/
6. Selesai!

=====================================
TIPS:
=====================================
- Pastikan nama file tidak ada spasi (gunakan - atau _)
- Format yang didukung: .jpg, .jpeg, .png, .webp, .gif
- Ukuran file sebaiknya tidak lebih dari 500KB
- Ukuran gambar yang ideal: 500x500 pixel (persegi)
