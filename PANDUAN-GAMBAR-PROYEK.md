2. Drag gambar dari folder download/sumber ke folder projects

## Spesifikasi Gambar yang Disarankan

- **Format:** JPG atau PNG
- **Ukuran:** Minimal 800x450 pixels (rasio 16:9)
- **Ukuran File:** Maksimal 2MB per gambar (agar loading cepat)
- **Kualitas:** Medium-High (tidak perlu terlalu besar)

## Tips Mendapatkan Gambar

1. **Screenshot dari video Anda sendiri**
   - Buka video di editor

   - Save sebagai JPG

2. **Dari project files**

   - Screenshot dari OBS Studio

3. **Placeholder sementara**
   - Gunakan gambar gaming/editing apapun dulu
   - Nanti bisa diganti dengan project asli

## Troubleshooting

**Gambar tidak muncul?**
- Cek nama file harus PERSIS sama (huruf kecil semua)
- Cek ekstensi file (.jpg bukan .jpeg)
- Cek lokasi folder sudah benar
- Hard refresh browser (Ctrl + Shift + R)

**Gambar terlalu besar/kecil?**
- CSS sudah mengatur ukuran otomatis
- Gambar akan di-crop sesuai rasio kartu

**Ingin ganti gambar?**
- Tinggal replace file dengan nama yang sama
- Refresh browser
