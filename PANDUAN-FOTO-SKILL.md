# 📸 Panduan Upload Foto Skill untuk Modal

## Struktur Folder

Buat folder baru untuk menyimpan foto skill:
```
portopolio/
├── images/
│   ├── skills/           ← FOLDER BARU
│   │   ├── premiere-pro.jpg
│   │   ├── photoshop.jpg
│   │   ├── illustrator.jpg
│   │   ├── lightroom.jpg
│   │   ├── after-effects.jpg
│   │   ├── davinci-resolve.jpg
│   │   └── canva.jpg
│   ├── profile/
│   ├── projects/
│   └── backgrounds/
```

## Cara Upload Foto Skill

### 1. Buat Folder Skills
1. Buka folder `images/` di project Anda
2. Buat folder baru bernama `skills`
3. Path lengkap: `D:/jule/portopolio/images/skills/`

### 2. Siapkan Foto untuk Setiap Skill

**Rekomendasi Ukuran Foto:**
- **Lebar**: 600-800px
- **Tinggi**: 400-600px
- **Rasio**: 4:3 atau 3:2 (landscape)
- **Format**: JPG atau PNG
- **Ukuran File**: Maksimal 500KB

**Jenis Foto yang Cocok:**
- Screenshot interface software yang sedang digunakan
- Hasil karya editing (before/after)
- Workspace setup dengan software terbuka
- Logo software dengan background menarik
- Kombinasi hasil karya + logo software

### 3. Nama File yang Harus Digunakan

| Skill | Nama File | Contoh Foto |
|-------|-----------|-------------|
| Adobe Premiere Pro | `premiere-pro.jpg` | Screenshot timeline editing |
| Adobe Photoshop | `photoshop.jpg` | Screenshot photo editing |
| Adobe Illustrator | `illustrator.jpg` | Screenshot vector design |
| Adobe Lightroom | `lightroom.jpg` | Screenshot color grading |
| After Effects | `after-effects.jpg` | Screenshot motion graphics |
| DaVinci Resolve | `davinci-resolve.jpg` | Screenshot color panel |
| Canva | `canva.jpg` | Screenshot design template |

### 4. Upload Foto

**Cara 1 - Copy Paste:**
1. Siapkan 7 foto dengan nama yang sesuai
2. Copy semua foto
3. Paste ke folder `images/skills/`

**Cara 2 - Drag & Drop:**
1. Buka folder `images/skills/` di File Explorer
2. Drag foto dari folder lain ke folder skills
3. Rename sesuai nama yang dibutuhkan

### 5. Verifikasi Upload

Pastikan struktur folder seperti ini:
```
images/skills/
├── premiere-pro.jpg      ✓
├── photoshop.jpg         ✓
├── illustrator.jpg       ✓
├── lightroom.jpg         ✓
├── after-effects.jpg     ✓
├── davinci-resolve.jpg   ✓
└── canva.jpg             ✓
```

## Tips Foto yang Bagus

### ✅ DO (Lakukan):
- Gunakan screenshot berkualitas tinggi
- Pastikan foto terang dan jelas
- Gunakan foto yang menunjukkan keahlian Anda
- Crop foto agar fokus pada bagian penting
- Gunakan foto dengan warna yang menarik

### ❌ DON'T (Jangan):
- Foto blur atau pixelated
- Foto terlalu gelap
- Foto dengan watermark orang lain
- Foto yang tidak relevan dengan skill
- Foto dengan ukuran file terlalu besar (>1MB)

## Jika Belum Ada Foto

Jika belum sempat upload foto, modal akan tetap berfungsi dengan:
- Icon besar di tempat foto
- Gradient overlay yang menarik
- Hover effect yang smooth

Foto akan muncul otomatis setelah Anda upload dengan nama file yang benar!

## Troubleshooting

### Foto Tidak Muncul?
1. **Cek nama file** - Harus persis sama (huruf kecil, pakai dash)
2. **Cek lokasi** - Harus di folder `images/skills/`
3. **Cek format** - Harus JPG atau PNG
4. **Refresh browser** - Tekan Ctrl+Shift+R (hard refresh)
5. **Clear cache** - Ikuti panduan di CARA-CLEAR-CACHE.md

### Foto Terlalu Besar/Kecil?
- Foto akan otomatis di-resize oleh CSS
- Tapi lebih baik resize manual ke 600-800px lebar
- Gunakan tool online: tinypng.com untuk compress

### Foto Terpotong Aneh?
- Gunakan rasio 4:3 atau 3:2
- Crop foto sebelum upload
- Pastikan bagian penting di tengah foto

## Update Setelah Upload

Setelah upload foto:
1. **Save** semua perubahan
2. **Refresh** browser (Ctrl+Shift+R)
3. **Test** dengan klik skill card
4. **Lihat** apakah foto muncul di modal

Foto akan muncul di sisi kiri modal dengan:
- Border radius 15px
- Hover effect dengan gradient overlay
- Icon besar muncul saat hover
- Sticky position saat scroll
